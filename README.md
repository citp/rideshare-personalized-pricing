V1 of uber personalized price aggregator

Instructions:
- Download the all the files under rideshare-personalized-pricing/ into a local directory
- Go to chrome://extensions/
- Load unpacked the directory
- You will be prompted to log into uber if you're not logged in
- Rides will be searched every 10 minutes until midnight UTC, and then it will stop
- The CSV with the price data will be available to download after midnight (or partial data earlier)
